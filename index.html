1. ¿Qué es la Diagonalización?

Es el proceso de transformar una matriz complicada en una matriz diagonal (simple).

  • Una Matriz Normal mezcla las coordenadas (x, y, z interactúan).
  • Una Matriz Diagonal actúa independientemente en cada eje.
  • Es fundamental para simplificar cálculos, como potencias de matrices (A^100).
"Cambiar a un sistema de coordenadas donde la transformación es simple."

2. La Fórmula Mágica

Una matriz $A$ es diagonalizable si existe una matriz $P$ tal que:

$$ A = P \cdot D \cdot P^{-1} $$
  • $D$ (Diagonal): Contiene los Valores Propios ($\lambda$) en la diagonal.
  • $P$ (Paso): Sus columnas son los Vectores Propios ($v$).
  • $P^{-1}$: Es la inversa de la matriz de paso.

3. El Algoritmo (Paso a paso)

Para hacerlo manualmente seguimos estos 3 pasos:

  1. Hallar Valores Propios ($\lambda$):
    Resolver $\det(A - \lambda I) = 0$.
  2. Hallar Vectores Propios ($v$):
    Resolver el sistema $(A - \lambda I)v = 0$ para cada $\lambda$.
  3. Construir Matrices:
    Formar $D$ con los $\lambda$ y $P$ con los vectores $v$.

4. Implementación en Python

Usamos la librería NumPy para automatizar el cálculo.


                import numpy as np

def demostrar_diagonalizacion():
    print("--- 1. DEFINIR LA MATRIZ A ---")
    # Usaremos una matriz 2x2 simple para el ejemplo
    # A = [[4, 1],
    #      [2, 3]]
    A = np.array([[4, 1], 
                  [2, 3]])
    print(f"Matriz Original A:\n{A}\n")

    print("--- 2. CALCULAR VALORES Y VECTORES PROPIOS ---")
    # np.linalg.eig hace todo el trabajo duro matemático
    valores_propios, vectores_propios = np.linalg.eig(A)

    print(f"Valores Propios (λ): {valores_propios}")
    print(f"Vectores Propios (Matriz P):\n{vectores_propios}\n")
    
    # NOTA: En 'vectores_propios', cada COLUMNA es un vector propio.

    print("--- 3. CONSTRUIR MATRICES D y P ---")
    # P es directamente la matriz de vectores propios que nos dio numpy
    P = vectores_propios
    
    # D debe ser una matriz diagonal con los valores propios
    D = np.diag(valores_propios)
    print(f"Matriz Diagonal D:\n{D}\n")
    
    print("--- 4. COMPROBACIÓN (A = PDP^-1) ---")
    # Calculamos la inversa de P
    P_inv = np.linalg.inv(P)
    
    # Realizamos la multiplicación P * D * P_inv
    # En Python, el operador '@' se usa para multiplicar matrices
    A_reconstruida = P @ D @ P_inv
    
    print(f"A reconstruida (P · D · P^-1):\n{A_reconstruida}")
    
    # Verificamos si es igual a la original (usando allclose por precisión decimal)
    es_igual = np.allclose(A, A_reconstruida)
    print(f"\n¿Es igual a la original? -> {es_igual}")

# Ejecutar la función
demostrar_diagonalizacion()
